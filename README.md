# Automated-Certificate-Generation-System
Python-based automation tool for bulk generation of student certificates using dynamic data input and customizable certificate templates.

Overview

A Python automation solution developed as a freelance project for an educational institution to automate bulk certificate generation.

The application reads student details from Excel sheets, automatically detects gender for correct pronoun usage, fills placeholders in a predefined Word template, and generates personalized certificates in bulk.

Features
Bulk certificate generation
Multi-sheet Excel processing
Automatic pronoun detection
Dynamic placeholder replacement
Folder organization by class or section
Word document generation
Reduced manual administrative workload
